StatForecast

A forecasting tool that uses historical sabermetrics and machine learning to estimate future player performance. This page links to the live app and the code repository.

Open Live App Repository

Methodology

I collected historical player-season and game-level sabermetrics (xwOBA, exit velocity, launch angle, strikeout rate, walk rate) and engineered features for trend, recent-form, and aging. I compared baseline baserate models with machine learning approaches.

Data

Data sources: Statcast CSV exports, Lahman for historical context, and play-by-play event datasets. I cleaned and aggregated to per-player seasonal summaries and created rolling windows for recent performance.

Models

I trained several models: ridge regression, random forest, XGBoost, and a small feed-forward neural network. Models were evaluated with cross-validation on time-based splits to avoid leakage.

Findings

Tree-based models with careful feature engineering outperformed linear baselines on mean absolute error for seasonal stat forecasts. Important features included age-adjusted plate appearances, recent month performance, and exit velocity percentiles.

Reproducibility

# clone
git clone https://github.com/christopherhsuu/statforecast
cd statforecast
# create and activate virtualenv, then install
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# run the Streamlit app
streamlit run app.py
