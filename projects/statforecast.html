StatForecast

A forecasting tool that uses historical sabermetrics and machine learning to estimate future player performance. This page links to the live app and the code repository.

Open Live App Repository

About the project

This project includes data ingestion, feature engineering, and several forecasting models (linear models, tree-based, and neural nets). The Streamlit app demonstrates the predictions and lets users explore inputs.

How to run locally

# clone
git clone https://github.com/christopherhsuu/statforecast
cd statforecast
# follow the project's README to set up a venv and install dependencies
