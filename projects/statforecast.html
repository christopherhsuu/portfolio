StatForecast

A forecasting tool that uses historical sabermetrics and machine learning to estimate future player performance. This page links to the live app and the code repository.

Launch (Live) Code
git clone https://github.com/christopherhsuu/statforecast

Overview (for non-experts)

StatForecast predicts near-term player performance using historical stats and recent trends. It's designed to give simple, explainable forecasts and show the features that drove predictions.

Who it's for

Fantasy managers, analysts, or anyone who wants a quick, explainable projection of a player's short-term outlook.

Quick start

  1. Open the live app.
  2. Search a player and pick a forecast horizon (week, month, season).
  3. Read the forecast and review the feature importance or recent-trend plots to understand why the model predicts that way.

Inputs & Outputs

Inputs: historical season/game stats and Statcast features. Outputs: numeric forecasts for stats (e.g., expected batting average, xwOBA) and visual explanations (feature importance, trend plots).

Methodology (technical)

I collected historical player-season and game-level sabermetrics (xwOBA, exit velocity, launch angle, strikeout rate, walk rate) and engineered features for trend, recent-form, and aging. Models compared baseline rates with ridge regression, random forest, XGBoost, and a small neural network; evaluation used time-based cross-validation to avoid leakage.

Reproducibility

# clone the repository
git clone https://github.com/christopherhsuu/statforecast
cd statforecast
# create and activate a virtualenv, then install
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# run the Streamlit app
streamlit run app.py
