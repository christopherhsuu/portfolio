About Hitter Archetypes

Hitter Archetypes is a clustering and visualization project that groups MLB hitters into meaningful profiles using Statcast and tracking-derived features. The goal is to combine modern tracking metrics (exit velocity, launch angle, bat speed, intercept offsets) with traditional contact and power statistics to reveal intuitive player archetypes — for example, high-exit-velocity power hitters, precise contact specialists, or speed-first hitters.

The pipeline begins by aggregating per-player features from pitch- and swing-level data. Aggregation scripts compute descriptive statistics (mean, median, p95) and sample-size metrics such as `n_swings`. Preprocessing handles missing values, winsorizes extreme percentiles, and expands circular angle variables into sin/cos components. These transforms stabilize downstream PCA and clustering and avoid artifacts from wrap-around angle values.

Clustering follows a standard, reproducible flow: select diagnostic features, preprocess (angle expansion, log1p for counts, winsorization, robust scaling), optionally reduce dimensionality with PCA for visualization, and run KMeans (the repo examples use k=5). The code exports cluster assignments and per-cluster CSVs for downstream analysis and saves transformation metadata (scalers, PCA components) so new players can be assigned to clusters consistently.

The Streamlit showcase is the user-facing piece: it loads the aggregated `data/player_archetypes.csv`, merges optional name mappings, and renders an interactive PCA scatter colored by cluster. Users can hover to inspect players, view centroid heatmaps, and read plain-English summaries generated from z-scored centroid values. The app also supports searching for a player and comparing that player's features against the cluster mean with special handling for circular features.

Practical considerations include choosing stable features, filtering by sample size to avoid small-sample artifacts, and fixing cluster `random_state` for reproducibility. Diagnostic checks (centroid heatmaps, representative players, silhouette scores) guide labeling of archetypes and tuning of k. Future work includes supervised prediction tasks using these archetype features, richer temporal aggregations, and a hosted interactive dashboard with server-side cluster assignment.

Hitter Archetypes is useful for scouts, analysts, and anyone who wants to categorize hitters in a reproducible, explainable way. The repo includes scripts, notebooks, and utilities to reproduce the clustering pipeline and to extend the analysis with additional tracking metrics or supervised models.
