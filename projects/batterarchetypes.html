Batter Archetypes

An exploratory clustering of hitters to identify archetypes (power/speed/contact/etc.) and visualize archetype profiles.

Launch (Live) Code

Overview (for non-experts)

This project groups hitters into "archetypes" so you can compare players with similar profiles. Think of archetypes as labels like "Power Hitter" or "Contact Hitter" that summarize a player's typical strengths.

Who it's for

Anyone curious about how hitters compare: coaches, fantasy players, or newcomers who want an easy way to see which players share similar skills.

Quick start

  1. Open the live demo.
  2. Type a player's name into the search box to find their archetype.
  3. Click the archetype centroid to see the typical stat profile for that group.

Inputs & Outputs

Inputs: seasonal batting metrics and Statcast features (e.g., ISO, strikeout rate, exit velocity). Outputs: archetype assignment for each player and centroid profiles that summarize each archetype.

How I built it (technical)

  1. Data ingest: batting metrics + Statcast for speed and exit velocity.
  2. Feature engineering: percentiles and rate metrics; imputed missing values for players with limited playing time.
  3. Clustering: standardized features, ran k-means, inspected silhouette scores and centroid stability.
  4. Deployment: Streamlit app to explore clusters and player membership.
