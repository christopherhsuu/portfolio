About StatForecast

StatForecast is a baseball forecasting tool built with R Shiny. It uses sabermetric indicators (xERA, barrel %, launch angle, etc.) to project player performance in a flexible, interactive interface.

This project reflects my interest in applying data science methods to real-world sports contexts, particularly forecasting and uncertainty in baseball analytics.
